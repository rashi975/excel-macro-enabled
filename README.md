# excel-macro-enabled
This Excel Macro-Enabled Sales Dashboard offers an interactive view of sales performance across regions and sales executives. It uses slicers, pivot tables, charts, and VBA automation to analyze total sales, target achievement, and performance gaps, enabling quick insights through easy filtering and dashboard navigation.
